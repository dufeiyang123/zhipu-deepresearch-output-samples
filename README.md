## 📖 Overview

This repository collects output samples and evaluation results from [Zhipu DeepResearch](https://research-hb.zhipuai-infra.cn/), evaluated on the [DeepResearch Bench](https://github.com/Ayanami0730/deep_research_bench) benchmark. Results have been submitted to the [DeepResearch Bench Leaderboard](https://huggingface.co/spaces/muset-ai/DeepResearch-Bench-Leaderboard).

Zhipu DeepResearch is a deep research agent that operates a two-phase pipeline — Plan (collaborative research & structured outline) and Write (deterministic per-chapter drafting with automated validation) — powered by multi-model backends and MCP-integrated web search and vision tools, delivering publication-ready 30,000+ word reports in 2–4 hours.
