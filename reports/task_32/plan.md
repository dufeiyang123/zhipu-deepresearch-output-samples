# Section 1：章节研究计划

## Chapter 1：导论——研究背景、综述范围与方法论说明
### 研究目标
- 阐明1937-1949年（抗日战争与解放战争）在中国近代史学科体系中的核心地位及其学术史梳理的必要性
- 界定本报告的综述范围：时间跨度（重点回顾1980年代以来、聚焦2010-2026年的学术成果）、学科边界（以大陆历史学界为主体，兼及港台与海外华人学者的相关成果）、文献类型（专著、学术论文、档案资料集、学位论文等）
- 说明横向对比分析的方法论框架：如何从研究领域、研究视角、研究方法、理论运用、研究结论等维度展开系统比较
- 简要勾勒1937-1949年研究的整体学术版图，为后续各章提供全景式导引

### 关键发现
- 1937-1949年涵盖抗日战争与解放战争两大历史时段，是中国从半殖民地半封建社会走向新中国的关键转折期。荣维木指出"抗日战争历史在中国近代历史中占据着重要地位"，战争结果"促成了中国近代历史因中华人民共和国的建立而告结束"[荣维木《近十年来抗日战争研究述评》](http://jxyyj.ruc.edu.cn/CN/abstract/abstract13825.shtml "《教学与研究》2005年第8期，第64-70页")
- 学术发展可分四阶段：①1949-1978年革命史框架期；②1978-1991年改革开放视野拓展期；③1991-2015年"一会一刊"创建与学科独立期；④2015年至今国家战略推动与全面深化期。高士华指出改革开放后"判断是非的标准，逐渐由以阶级意识为重转向以民族大义为重"[高士华《抗日战争研究在改革开放中不断发展》](http://jds.cass.cn/xscg/xslw/201811/t20181126_5253004.shtml "《人民日报》2018年11月26日第22版")
- 1991年中国抗日战争史学会成立、《抗日战争研究》创刊（CSSCI来源期刊，全国唯一抗战史专业学术期刊），标志着抗战研究"从党史、中国革命史和民国史研究中独立出来，成为一个相对独立的研究领域"[《抗日战争研究》编辑部简介](http://jds.cass.cn/bsgk/zzjg/krzzyjbjb/ "近代史研究所官网")
- 2015年中央政治局第25次集体学习，习近平强调"我们的抗战研究还远远不够，要继续进行深入系统的研究"，此后国家社科基金设立抗日战争研究专项工程（2016年起），至今立项30余项、出版100余种资料集[新华网](http://www.xinhuanet.com/politics/2015-12/10/c_128518232.htm "2015年重要论述") [吴敏超《近十年中国抗战史研究的蓬勃发展》](http://hprc.cssn.cn/gsyj/zhutiyj/kz80/202506/t20250619_5880073.html "《中国社会科学报》2025年6月19日")
- 2016年中国抗日战争研究协同创新中心在南京大学揭牌，由南京大学、北京大学、南开大学、武汉大学牵头，协同中国社科院近代史研究所、中央档案馆、中国第二历史档案馆等共建[南京大学官网](https://www.nju.edu.cn/info/3191/178381.htm "2016年1月21日")
- 2018年"抗日战争与近代中日关系文献数据平台"正式上线，截至2024年底总文献数达7400万余页，为"亚洲最大的中国抗日战争史数据库"[全国哲学社会科学工作办公室](http://www.nopss.gov.cn/n1/2018/0903/c219468-30269215.html "2018年9月3日") [吴敏超2025综述](http://hprc.cssn.cn/gsyj/zhutiyj/kz80/202506/t20250619_5880073.html "《中国社会科学报》2025年6月19日")
- 2019年步平、王建朗主编《中国抗日战争史》（全8卷）出版，"是迄今中国抗日战争史研究领域最大规模、最具权威性的综合性研究专著"[吴敏超2025综述](http://hprc.cssn.cn/gsyj/zhutiyj/kz80/202506/t20250619_5880073.html "《中国社会科学报》2025年6月19日")
- 2019年5月近代史研究所设立抗日战争史研究室，2023年该学科入选中国社会科学院"登峰战略"重点学科[近代史研究所抗战史研究室简介](http://jds.cssn.cn/bsgk/zzjg/krzzsyjs/ "近代史研究所官网")
- 2025年纪念抗战胜利80周年，多场国际学术研讨会举行；2026年《抗日战争研究》将由季刊改为双月刊[南京大学官网](https://www.nju.edu.cn/info/1056/417891.htm "《社科院专刊》总第762期")
- 国家档案局组织推出《抗日战争档案汇编》计划出版1000册，截至2023年底已出版550余册[吴敏超2025综述](http://hprc.cssn.cn/gsyj/zhutiyj/kz80/202506/t20250619_5880073.html "《中国社会科学报》2025年6月19日")
- 核心期刊：《抗日战争研究》（1991创刊）、《近代史研究》（1979创刊）、《历史研究》（1954创刊）、《日本侵华南京大屠杀研究》
- 主要学术机构：中国社科院近代史研究所（1950年成立）、南京大学中华民国史研究中心（1993年成立，教育部首批人文科学重点研究基地）、中国抗日战争研究协同创新中心（2016年成立）、中国抗日战争史学会（1991年成立）、中国社科院中日历史研究中心
- 已有权威综述文章：荣维木《近十年来抗日战争研究述评》（2005）、高士华《抗日战争研究在改革开放中不断发展》（2018）、吴敏超《近十年中国抗战史研究的蓬勃发展》（2025）、吴敏超《中国共产党抗战史研究的趋势、特点与相关思考》（2026）[近代史研究所](http://jds.cass.cn/xscg/xslw/202601/t20260130_5971445.shtml "2026年1月30日")

# Section 2：给 Write 阶段的执行建议
- 全文应以"横向对比"为核心方法论贯穿始终，每个维度的分析均需有明确的比较对象、比较标准和比较结论，避免沦为简单的研究成果罗列或流水账式综述
- 注意"抗日战争"（1937-1945）与"解放战争/国共内战"（1946-1949）两个子时段的平衡处理：抗战史研究体量远大于解放战争研究，报告应如实反映这一不均衡格局并分析其原因，而非人为拉平篇幅
- 范式转换的叙述应避免简单的线性进化论述，需呈现不同范式在同一时段的并存、竞争与交互影响关系
- 代表性论著的选择应兼顾经典奠基之作（如张宪文、胡德坤等人的系统性著作）和近年突破性成果，避免仅列老一代学者成果
- "未来研究潜力选题"章节需严格基于前文分析得出，论证逻辑需从"现有空白/不足→可用资源/条件→预期学术贡献"三个层面展开，确保选题建议的说服力
- 关于期刊和学术共同体的讨论（如《抗日战争研究》《近代史研究》《民国研究》等核心期刊的引领作用、中国抗日战争史学会等学术组织的推动）可适当融入各章，但不必单列一章，以免打断分析主线
- 涉及海外（含港台）学术成果时，重点关注其对大陆学界的影响和对话关系，而非面面俱到地综述海外研究
- 行文应保持学术报告的严谨性与客观性，在评价不同研究范式和学术观点时避免价值预判，应以学术贡献和证据力度作为评判标准
- 时间口径：研究对象为1937-1949年中国历史；学术综述重点回顾1980年代至2026年4月的研究成果，聚焦2010年以来的最新进展
